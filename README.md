# archive-cnofficial-leyu

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式保存，可根据需要直接访问或用作资料参考。仓库本身不针对任何单个域名或具体网站，仅作为通用存档用途。

## 目录说明

```
/
├── index.html         # 仓库主页 / 导航页（如有）
├── pages/             # 存放主要归档 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/            # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md          # 本文件
```

- `pages/`：归档的 HTML 页面存放目录。文件名按顺序或标识命名，无特定规律。
- `assets/`：页面所需的公共资源文件，按类型分目录存放。

## 页面归档说明

- 所有归档页面均为独立 HTML 文件，不依赖外部服务或后端。
- 页面内容可能来源于多个不同时间点的快照或手工整理，仅供存档与查阅。
- 部分页面可能包含内联样式或脚本，以确保离线或静态环境下可正常显示。
- 页面内容可能未经过实时更新或验证，请以原始来源为准。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接用浏览器打开 `pages/` 目录下的任意 `.html` 文件即可查看。
3. 若需要部署到静态托管服务（如 GitHub Pages），可将仓库根目录作为站点根目录。
